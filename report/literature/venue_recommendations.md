# Venue Recommendations for RouteSmith Paper

## Tier 1: Stretch Venues (Acceptance Rate ~20-25%)

### **NeurIPS 2026** — Track: AI Systems / Machine Learning Systems
- **Deadline:** May 15, 2026 (abstract), May 22, 2026 (full paper)
- **Fit:** ⭐⭐⭐⭐⭐ RL + Systems intersection is perfect
- **Why:** 
  - NeurIPS has strong ML systems track (vLLM, TGI papers accepted recently)
  - Multi-armed bandits are well-represented

- **Acceptance rate:** ~25% (2025: 26.3%)
- **Reviewer profile:** Mix of RL theorists and systems researchers
- **Risk:** High — needs strong empirical results + theoretical contributions

- **Deadline:** January 23, 2026 (abstract), January 30, 2026 (full paper)
- **Fit:** ⭐⭐⭐⭐⭐ Multi-armed bandits application
- **Why:**
  - ICML loves bandit papers (Thompson Sampling specifically)

  - Strong industry presence (good for LLM cost topic)
- **Acceptance rate:** ~27% (2025: 27.2%)
- **Reviewer profile:** RL experts, bandit theorists
- **Risk:** High — needs theoretical novelty beyond application
- **Recommendation:** Strong contender if you emphasize TS convergence analysis

### **ICLR 2026** — Track: Deep Learning Systems / RL
- **Deadline:** September 30, 2025 (abstract), October 7, 2025 (full paper)
- **Fit:** ⭐⭐⭐⭐ LLM + RL combination

## Tier 3: Safe Venues (Acceptance Rate ~40-60%)

### **arXiv Preprint** — cs.LG + cs.CL
- **Timing:** Immediate (no review)
- **Fit:** ⭐⭐⭐⭐⭐ Establish priority
- **Why:**
  - No review delay — establish priority immediately
  - Citable and permanent
  - Can submit to conferences later
  - Many LLM routing papers start as arXiv preprints
- **Strategy:** Post to arXiv first, then submit to EMNLP/ACL
- **Categories:** cs.LG (Machine Learning), cs.CL (Computation and Language)
- **Recommendation:** **DO THIS FIRST** regardless of conference plans

### **TMLR (Transactions on Machine Learning Research)**
- **Deadline:** Rolling submissions
- **Fit:** ⭐⭐⭐⭐ Strong ML methods
- **Why:**
  - FrugalGPT published here (Chen et al., 2024)
  - Rigorous review but no hard deadlines
  - Open review process
  - High visibility in ML community
- **Acceptance rate:** ~40-50%
- **Reviewer profile:** ML researchers across subfields
- **Risk:** Low — rolling submission allows iterations
- **Recommendation:** Excellent venue if you want journal-style publication

### **JMLR (Journal of Machine Learning Research)**
- **Deadline:** Rolling submissions
- **Fit:** ⭐⭐⭐ Theoretical contributions needed
- **Why:**
  - Prestigious journal
  - Bandit theory well-represented
  - Long review process (6-12 months)
- **Acceptance rate:** ~30-40%
- **Risk:** Medium — needs stronger theoretical contributions
- **Recommendation:** Only if you add theoretical TS analysis

## Conference Formatting Notes

- **NeurIPS/ICML/EMNLP/ACL:** Use official LaTeX templates
- **TMLR:** JMLR format
- **arXiv:** Standard ACM or NeurIPS format
- **Page limits:** 
  - Main conferences: 9-10 pages (excluding references)
  - TMLR: No strict limit
  - Workshops: 6-8 pages

**Important:** ArXiv posting does NOT count as prior publication for most ML conferences (NeurIPS, ICML, EMNLP, ACL all allow it). Always verify on conference website.
