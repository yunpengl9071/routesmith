\subsection{Reproducibility}
The existing \texttt{origin/feature/contextual-bandit-routing} branch contained unverified and partially simulated results from an earlier prototype. All results in this paper are from real API calls recorded as JSON files in \texttt{benchmark/results/} with fixed seeds. The JSON files are not committed to git (size), but the experiment scripts and random seeds are fully reproducible.

\textbf{RouteLLM-SW fallback:} If the RouteLLM SW router fails to load due to embedding dimension mismatch (arena embeddings are 1536-dim; OpenRouter does not expose text-embedding-3-small), we fall back to \texttt{all-MiniLM-L6-v2} (384-dim). Results using this fallback are marked with $\dagger$. The cosine similarity computation remains valid, but absolute magnitudes differ from the original arena-embedding space.
